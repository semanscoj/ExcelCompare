#REDUCE
	Returns all rows from A that are missing from B
	optionally filter rows in A before subtraction
#UNION
	Returns all of the rows from A
	Highlights all Rows in A that don't appear in B
	Rows unique to B will NOT appear in A
#FILL
	Adds a column to B using A as a reference
#HELP
	Please refer to the built in help when running the application.
